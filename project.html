BusinessBrain AI — সিস্টেম আর্কিটেক্চার

ইউজারের প্রাকৃতিক ভাষার কমান্ড কীভাবে পুরো সিস্টেমের ভেতর দিয়ে প্রবাহিত হয়ে রেজাল্ট হয়ে ফিরে আসে — উপর থেকে নিচে।

Layer 0 · User
ইউজার — প্রাকৃতিক ভাষায় কমান্ড (বাংলা / ইংরেজি)
যেমন: "Analyze sales decline" বা "নিষ্ক্রিয় কাস্টমারদের সেগমেন্ট বানাও"
↓
Layer 1 · Frontend — Next.js + TypeScript
চ্যাট ইন্টারফেস + ড্যাশবোর্ড UI
Phase 5 স্কিল দিয়ে তৈরি — CRM ভিউ, KPI ড্যাশবোর্ড এবং AI চ্যাট প্যানেল একসাথে।
↓
Layer 2 · API Gateway — FastAPI
Auth (JWT/OAuth2) · Rate Limiting · Request Routing
Phase 3 ও Phase 10 স্কিল দিয়ে তৈরি — প্রতিটি রিকোয়েস্ট এখানে যাচাই হয়ে সঠিক সার্ভিসে যায়।
↓
